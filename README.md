# Evaluating Virtual Outcrop Models (VOMs) in Geoscience Education

## Overview
This repository serves as the central hub for a research project investigating the pedagogical effectiveness of Virtual Outcrop Models (VOMs). It tracks the study, from initial literature review to  final analysis.

## Project Scope & Objectives
This project aims to understand how students interact with and learn from VOMs compared to traditional physical fieldwork. The core objectives include:

* **Literature Review:** Aggregating and synthesizing existing research on the use and evaluation of VOMs for teaching purposes.
* **Assessment Development:** Designing and validating instruments to measure student learning outcomes with and without the use of VOMs.
* **Investigation Design:** Structuring a comprehensive study to observe and quantify how students use VOMs for learning.
* **Field Logistics:** Coordinating volunteer recruitment and managing physical field site logistics (transportation, safety protocols, and catering).
* **Data & Analysis:** Collecting student performance data, conducting statistical analysis, and drafting manuscripts to report findings.

## Repository Structure


* `/literature-review` - Reference libraries, summaries, and manuscript drafts.
* `/assessment-instruments` - Pre-test/post-test materials, surveys, and evaluation rubrics.
* `/study-design` - Methodology outlines, ethics/IRB approvals, and volunteer recruitment materials.
* `/field-logistics` - Itineraries, safety waivers, bus schedules, and catering lists.
* `/data-analysis` - Raw data (anonymized), analysis scripts (e.g., Python/R), and visualization outputs.
* `/publications` - Drafts of articles and reports intended for publication.
